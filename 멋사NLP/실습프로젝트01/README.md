# 실습프로젝트01: `긴급도 판별`

## 데이터 및 과제

### 태스크

접수된 신고의 긴급도를 판별하는 과제입니다.

### 난이도

★★★☆☆(음성-텍스트 멀티 모달이 까다로울 수 있음)

### 제한 사항

음성 파일(.wav)과 발화된 음성을 변환한 텍스트를 사용해 긴급도(urgencyLevel)와 감정 상태(sentiment)를 판별하는 멀티 모달리티, 멀티 태스크 모델을 구현하십시오.

### 학습 데이터

위급상황 음성/음향 (고도화) - 119 지능형 신고접수 음성 인식 데이터[[링크]](https://www.aihub.or.kr/aihubdata/data/view.do?currMenu=115&topMenu=100&aihubDataSe=data&dataSetSn=71768)
